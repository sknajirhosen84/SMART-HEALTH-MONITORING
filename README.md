# SMART-HEALTH-MONITORING 
The Integrated Smart Health Monitoring & Medicine Reminder Assistant is a dual‑function, IoT‑based device designed to address two critical healthcare needs: continuous, real‑time patient vital signs monitoring and intelligent medication adherence support. Health monitoring part: Uses an ESP32 microcontroller, multiple biomedical sensors (MAX30102 for HR/SpO₂, AD8232 for ECG, MPU6050 for position, DHT11/DS18B20 for temperature/humidity, NEO‑6M GPS, SIM800L GSM), a 1.8‑inch TFT display, and a Firebase cloud database. It measures heart rate, blood oxygen saturation, body and ambient temperature, humidity, patient posture (roll/pitch), live ECG waveform, and GPS location. Data is sent to Firebase and displayed on a web dashboard (HTML/CSS/JS, Chart.js, Leaflet.js) with real‑time graphs, an interactive map, emergency alerts, and a historical data viewer. An emergency button triggers an SMS with a Google Maps link.
· Medicine reminder part: Uses an ATmega328P standalone microcontroller (Arduino core), an OLED display (128×64, I²C), a DS3231 RTC module, four push buttons (Up, Down, OK, Back), a buzzer, and a red LED. Users can add, edit, delete, and view medicine schedules through a simple on‑screen menu. All reminders are stored in the ATmega328P’s EEPROM (non‑volatile, survives power loss). At the scheduled time, the buzzer sounds, the LED flashes, and the OLED displays the medicine name. Pressing any button stops the alarm.
